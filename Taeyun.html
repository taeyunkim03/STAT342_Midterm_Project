Z-Test & T-Test

One Sample Design

Model: \(X_1, X_2, \ldots, X_n \overset{i.i.d.}{\sim} f\) where \(E[X] = \mu_0\) and \(SD[X] = \sigma_0\)

Parameter of Interest: \(\mu_0 = E[X]\)

Point Estimator: \(\hat{\mu}_0 = \bar{X}\)

Z-Test

One Sample Mean Inference (Large Sample)

Central Limit Theorem:
\[\bar{X} \approx \text{Norm}\left(\mu_0, \frac{\sigma_0}{\sqrt{n}}\right)\] Equivalently: \[Z = \frac{\sqrt{n}(\bar{X} - \mu_0)}{\sigma_0} \approx \text{Norm}(0, 1)\]

Confidence Interval

100(1-α)% Confidence Interval for μ₀: \[\bar{X} \pm z_{\alpha/2} \frac{\sigma_0}{\sqrt{n}}\]

Note: when \(\sigma_0\) is unknown, replace it with a consistent estimate.

Hypothesis Testing

Hypotheses:
\(H_0: \mu_0 = \mu_0^{null}\)
\(H_1: \mu_0 \neq \mu_0^{null}\) (two-sided)

Test Statistic:

\[Z_0 = \frac{\sqrt{n}(\bar{X} - \mu_0^{null})}{\sigma_0} \approx \text{Norm}(0, 1)\]
Alternative P-value
\(H_1: \mu_0 > \mu_0^{null}\) \(P(Z \geq z_0)\)
\(H_1: \mu_0 < \mu_0^{null}\) \(P(Z \leq z_0)\)
\(H_1: \mu_0 \neq \mu_0^{null}\) Double 1-sided P-value

Binomial Proportion

For success probability \(\pi_0\), testing \(H_0: \pi_0 = \pi_0^{null}\):

Sampling Distribution: \[\bar{X} \approx \text{Norm}\left(\pi_0, \sqrt{\frac{\pi_0(1-\pi_0)}{n}}\right)\] Confidence Interval: \[\bar{X} \pm z_{\alpha/2}\sqrt{\frac{\bar{X}(1-\bar{X})}{n}}\] Test Statistic: \[Z = \frac{\sqrt{n}(\bar{X} - \pi_0^{null})}{\sqrt{\pi_0^{null}(1-\pi_0^{null})}}\]

Duality

In testing \(H_0: \mu_0 = \mu_0^{null}\) versus \(H_1: \mu_0 \neq \mu_0^{null}\) using the z-test, when \(H_0\) is rejected at the α-level, the 100(1-α)% confidence interval does not contain \(\mu_0^{null}\).

T-Distribution

When \(\sigma_0\) is unknown and replaced with sample standard deviation \(S\), the ratio:

\[T = \frac{\sqrt{n}(\bar{X} - \mu_0)}{S}\]

does not have a standard normal distribution when \(n\) is small.

Properties

  • Bell-shaped and symmetric about 0
  • Characterized by degrees of freedom: df = n - 1
  • Thicker tails than the normal distribution
  • Approaches standard normal as df increases
Theorem: If \(X_1, X_2, \ldots, X_n \overset{i.i.d.}{\sim} \text{Norm}(\mu_0, \sigma_0)\), then: \[T = \frac{\sqrt{n}(\bar{X} - \mu_0)}{S} \sim t_{n-1}\] where \(S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2\)

The One-Sample T-Test

Assumptions

Data: \(X_1, X_2, \ldots, X_n \overset{i.i.d.}{\sim} \text{Norm}(\mu_0, \sigma_0)\) both unknown
No restrictions on sample size \(n\)

Confidence Interval

100(1-α)% Confidence Interval for μ₀: \[\bar{X} \pm t_{\alpha/2, n-1} \frac{S}{\sqrt{n}}\]

Hypothesis Testing

Hypotheses:
\(H_0: \mu_0 = \mu_0^{null}\)

Test Statistic:

\[T_0 = \frac{\sqrt{n}(\bar{X} - \mu_0^{null})}{S} \sim t_{n-1}\]
Alternative P-value
\(H_1: \mu_0 > \mu_0^{null}\) \(P_{\mu_0=\mu_0^{null}}(T_0 \geq t_0)\)
\(H_1: \mu_0 < \mu_0^{null}\) \(P_{\mu_0=\mu_0^{null}}(T_0 \leq t_0)\)
\(H_1: \mu_0 \neq \mu_0^{null}\) Double 1-sided P-value

Duality

If \(\mu_0^{null}\) is an interior point of the 100(1-α)% confidence interval, then the P-value will be larger than α.

Paired Data

Definition: Measurements taken in pairs (e.g., twins, same subject before and after treatment) to keep conditions constant and reduce confounding.

Notation: \((X_1, Y_1), (X_2, Y_2), \ldots, (X_n, Y_n)\)

Covariance

Definition: \[\text{Cov}[X, Y] = E[(X - E[X])(Y - E[Y])]\] Alternative: \[\text{Cov}[X, Y] = E[XY] - E[X]E[Y]\]

Note: In paired designs, \(X_i\) and \(Y_i\) are typically positively associated. Independence implies zero covariance, but zero covariance does not imply independence.

Analysis of Differences

Transform pairs into a single sample of differences:

\[D = X - Y\]
Hypotheses:
\(H_0: \mu_d = 0\)

Test Statistic:

\[T_0 = \frac{\sqrt{n}(\bar{D} - \mu_d^{null})}{S_D}\] where \(S = \frac{1}{n-1}\sum_{i=1}^{n}(D_i - \bar{D})^2\)

Summary Comparison

Test Assumption Test Statistic Distribution
Z-test Large n, any distribution \(\frac{\sqrt{n}(\bar{X} - \mu_0^{null})}{\sigma_0}\) Norm(0, 1)
One-sample T-test Normal distribution \(\frac{\sqrt{n}(\bar{X} - \mu_0^{null})}{S}\) \(t_{n-1}\)
Paired T-test Paired data \(\frac{\sqrt{n}(\bar{D} - \mu_d^{null})}{S_D}\) \(t_{n-1}\)
